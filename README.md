# Hackintosh-on-QEMU
This repository was created for the purpose of demo on agile teamwork only and is not intended to be used as a project for real work.

# Aim of this project
This project collects essential methods and related scripts, tools and whatever for a novice to deploy Apple's macOS on a normal pc based on qemu technology ***FOR DEMO OF AGILE TEAMWORK ONLY***
